# CNN

🔢 Handwritten Digit Recognition using CNN (MNIST)

This project implements a Convolutional Neural Network (CNN) to recognize handwritten digits (0–9) using the MNIST dataset.
It demonstrates deep learning fundamentals, image preprocessing, model training, and prediction.

🚀 Project Overview

🧠 CNN model for image classification

✍️ Recognizes handwritten digits from images

📊 Trained on MNIST dataset

💾 Saved trained models (.h5 files)

📓 Implemented using Jupyter notebooks

📂 Project Structure
MNIST-CNN/
│
├── CNN.ipynb                 # CNN model implementation
├── MNIST.ipynb               # Dataset preprocessing & training
├── mnist_cnn.h5              # Saved trained model
├── mnist_cnn_model.h5        # Final CNN model
└── sample digit images       # Test handwritten digit images
